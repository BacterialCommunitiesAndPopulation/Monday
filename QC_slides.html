Sequence quality

  • Not all NGS data is reliable, so QC is needed before downstream analysis

  • Sequencing platforms give quality values for each base

  • PHRED score (Q)
    • Error probability: P = 10-Q/10
    • Single letter ASCII characters, PHRED+33

Primers and adapters

  • PCR primers
    • Part of the amplicon
    • Normally need to be removed
  • Sequencing adapters
    • Library preparation
    • Outside amplicon
    • Should be automatically removed

Tools for QC and trimming

  • FastQC
    • Quality control tool
    • Easy to use, also interactve mode, nice graphical output
    • Mainly meant for shotgun sequencing


  • Cutadapt
    • Adapter + primer removal
    • Quality and length trimming
    • Works in paired-end mode
